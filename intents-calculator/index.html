Intents Calculator

To get gateway events you will need to provide a value to the GatewayClient so that it can subscribe to the set of events. You may get more information about these in our guide and the Discord documentation.

Below you can choose the intents or events you will need, and it will give the intent value to pass.

Intents marked in red and bold are privileged intents, or events related to these intents, that need to be enabled in the developer portal. Events in blue and bold are default events that do not require any intents and will always be dispatched.
